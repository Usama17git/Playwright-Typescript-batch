# Playwright TypeScript Batch

## Run tests

```powershell
npm run test
```

## Run tests in headed mode

```powershell
npm run test:headed
```

## Run tests and open Allure report automatically

```powershell
npm run test:allure
```

## Generate Allure report from existing results

```powershell
npm run allure:generate
```

## Open the generated Allure report

```powershell
npm run allure:open
```

## Serve Allure results directly

```powershell
npm run allure:serve
```

## Prerequisites

- Node.js and npm
- Java installed and `JAVA_HOME` configured for Allure
